# 🐾 SALOVET

## 📖 Descripción

**SALOVET** es una aplicación diseñada para la gestión integral de clínicas veterinarias. Permite administrar información de pacientes, propietarios, historiales clínicos y otros procesos esenciales de forma eficiente y centralizada.

Su objetivo principal es facilitar el trabajo diario del personal veterinario mediante una herramienta moderna, intuitiva y escalable.

---

## 🚀 Funcionalidades principales

* Gestión de pacientes (mascotas)
* Registro de propietarios
* Historial clínico
* Organización de citas
* Administración de datos veterinarios
* Asistencia Veterinaria con IA

---

## 🛠️ Tecnologías utilizadas

### 💻 Entorno de desarrollo

* Visual Studio

### ⚙️ Frameworks y plataformas

* .NET MAUI
* .NET Framework

### 👨‍💻 Lenguajes de programación

* C#
* SQL (MySQL)

---

## 🎯 Finalidad del proyecto

El proyecto SALOVET nace con la finalidad de digitalizar y optimizar la gestión en clínicas veterinarias, reduciendo el uso de papel y mejorando el acceso a la información en tiempo real.
